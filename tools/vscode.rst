VSCode
==============

.. contents::
    :local:
    :depth: 1


简介
------------

`VisualStudioCode <https://code.visualstudio.com/>`_ 是由微软研发的一款 `开源 <https://github.com/microsoft/vscode>`_ 的跨平台代码编辑器，目前是前端(网页)开发使用最多的一款软件开发工具。


功能
------------

快捷键
~~~~~~~~~~~

* 折叠所有区域代码的快捷： ctrl + k   ctrl + 0 ;( 注意这个是零，不是欧 )
* 展开所有折叠区域代码的快捷：ctrl +k  ctrl + J ;

调试常见的快捷键

* F9 打开和停止调试断点
* F11 单步调试
* F5 启动调试
